## 목적
- 전국 아파트 매물을 조회할 수 있는 서비스를 만든다.
- 법정동코드, 동이름, 현재 위치 기반으로 아파트를 조회한다.
- 매물 별 주변 시설 정보를 제공한다.

<br>

## 실행 방법
### 실행 환경: vue-board-bootstrap

#### setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
```
npm run lint
```
